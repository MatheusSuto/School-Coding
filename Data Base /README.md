# Data-Base

How to setup the database

// Start Xampp - start Apache -> MySql

// Open up MySql 

// Go to MySql Connections and create a new file using the + button to create a connection

// Test connection, even if it sayd that it isn't compatible, do it. 

// Go to schemas and right button on any whitespace there, create a new schema

// Name the schema, in chatset choose latin1, latin1_general_ci

// In the database, it will come out with a command as such: "CREATE SCHEMA `3c xavier` DEFAULT CHARACTER SET latin1 COLLATE latin1_general_ci ;", which is bascialy a script.

// Run it, get to Schemas and verify if it's there (*if it isn't there, reflesh*)

// Go to the upper left, click the file icon with SQL and a + symbol on it

// write the code: 
   USE `3c xavier`; 
// Or whatever the fuck you wrote there
